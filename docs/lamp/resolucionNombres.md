Para poder acceder a nuestro dominio creado (por ejemplo, www.ejemplo.com), precisamos que nuestro equipo "resuelva" dicho nombre de dominio, es decir, sea capaz de traducir a qué dirección IP apunta el mismo. 

Por ejemplo, la dirección IP correpondiente al dominio www.itel.edu.ar es 168.197.51.107.
