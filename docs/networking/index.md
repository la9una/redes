Una red informática tiene por misión interconectar equipos. Sin embargo, ante el crecimento de la misma, su administración puede resultar compleja y difícil de mantener. 

Para facilitar la ampliación de la red, podemos crear subredes, es decir, dividir la red en segmentos más pequeños para facilitar de este modo, su posterior gestión. 

Podemos crear subredes de dos modos: 

* Física
* Lógica

## Subred física
Es una división de la red empleando _routers_ para cada subred e interconectados entre sí.

## Subred lógica
Se trata de una división de la red, mediante la configuración basada en direcciones IP y máscara de red.
