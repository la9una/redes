
Presioná las teclas <kbd><i class="fab fa-windows"></i></kbd> + <kbd>R</kbd> para acceder al formulario **Ejecutar** donde podrás introducir los comandos. 

!!!done "Accesos directos a elementos del Panel de Control"
		Un archivo `.cpl` (abreviatura de _
		Control Panel Launcher_) no es otra cosa que un acceso directo hacia un elemento del Panel de Control de Windows. 

Conexiones de red

```
ncpa.cpl
```

Opciones de internet

```
inetcpl.cpl 
```

Firewall

```
firewall.cpl
```

Propiedades de sistema

```
sysdm.cpl 
```

Desinstalar o cambiar un programa

```
appwiz.cpl 
```

Administrador de dispositivos

```
hdwwiz.cpl
```

Fecha y hora de sistema

```
timedate.cpl 	
```
